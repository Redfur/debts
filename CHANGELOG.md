# Changelog

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.1.0/). В заголовке версии **без** префикса `v` (например `1.2.3`); в git тег задаётся с префиксом: `v1.2.3`.

Заметные изменения поведения и UX (новые сценарии, изменения экранов, хранилища под пользовательские действия) фиксируйте в разделе **[Unreleased]** в том же изменении, что и код — чтобы история релизов оставалась полной.

## [1.1.0] - 2026-07-02

### Added

- Страница «Контакты» (`/contacts`) со списком всех сохранённых контактов; на главной теперь только контакты с непогашенным долгом.
- Создание долга — отдельная страница `/new` вместо модалки.
- Изменение долга (направление/сумма/валюта — пока нет погашений; заметка — всегда) через новую модалку.
- Удаление долга доступно и для погашенных (закрытых) долгов, не только активных.
- Маска разделения разрядов в поле суммы долга/погашения (`10000` → `10 000`).
- Заметки к операциям (в т.ч. к погашению) отображаются в истории долга — свёрнутой по умолчанию, чтобы не перегружать карточку.

### Changed

- Навигация: три вкладки (Главная / Контакты / Настройки); активная вкладка на карточке контакта зависит от того, откуда на неё перешли.

### Fixed

- Открытие `Select` (или другого вложенного слоя) внутри модального окна больше не закрывает модалку при клике внутри её границ.
- Исправлен потенциальный бесконечный цикл ре-рендера при удалении долга, для которого ещё было открыто диалоговое окно (Zustand-селектор возвращал новый пустой массив на каждый вызов).

## [1.0.1] - 2026-07-02

### Changed

- Заменены favicon и иконки приложения (apple-touch-icon, favicon.ico/svg, web-app-manifest 192/512) на финальные.

## [1.0.0] - 2026-07-02

Первый стабильный релиз.

### Added

- Главная: итоги «мне должны» / «я должен» с конвертацией в валюту отображения и разбивкой по валютам, список контактов с нетто-остатком.
- Создание долга на контакт (поиск существующего или создание нового по имени), направление, сумма, валюта (RUB/USD/EUR/CNY), заметка.
- Погашение долга (частичное или полное) с валидацией остатка; автоматическое закрытие долга при остатке 0.
- Карточка контакта: сводка по направлениям, список долгов с прогрессом погашения и историей операций, удаление долга.
- Курс валют: автообновление не чаще раза в сутки + ручное обновление в настройках, офлайн-фолбэк на последний известный курс.
- Настройки: тема, валюта отображения, статус курса валют, резервная копия данных (экспорт/импорт JSON), сброс данных.
- PWA офлайн-режим (`vite-plugin-pwa`).
