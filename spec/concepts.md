# Термины и концепции

## Contact (контакт)

Человек, с которым связаны один или несколько долгов. Создаётся неявно при создании первого долга на него (по имени, без дублей — поиск без учёта регистра). У контакта — цвет аватара-инициала, детерминированный по имени.

## Debt (долг)

Одна запись долга: направление, валюта, исходная сумма (`principalAmount`), заметка, статус. Не редактируется после создания напрямую — баланс меняется только через операции погашения.

## DebtOperation (операция)

Событие в истории долга: `initial` (создание, сумма = `principalAmount`) или `repayment` (погашение, уменьшает остаток). Остаток долга = `principalAmount − Σ(repayment.amount)`.

## Направление (direction)

- `owed_to_me` — контакт должен пользователю («мне должны»).
- `i_owe` — пользователь должен контакту («я должен»).

## Валюта отображения (display currency)

Настройка в `/settings`, по умолчанию RUB. Используется только для конвертации агрегированных итогов на главной и в карточке контакта; сами долги хранятся и показываются в исходной валюте записи.

## Курс валют (ExchangeRatesCache)

Курсы всех поддерживаемых валют к одной базовой (USD), закэшированные локально с датой и временем загрузки. Обновляются не чаще раза в сутки; используются только для конвертации, не хранятся как часть домена долгов.

## Связи

```
Contact (1) ──< (N) Debt
                     │
                     └──< (N) DebtOperation ("initial" | "repayment")
```

- **ContactSummary** — производная от `Debt` + `DebtOperation` конкретного контакта: остатки по валютам, раздельно по направлению.
- **GlobalTotals** — то же самое по всем контактам + конвертация в валюту отображения через `ExchangeRatesCache`.
